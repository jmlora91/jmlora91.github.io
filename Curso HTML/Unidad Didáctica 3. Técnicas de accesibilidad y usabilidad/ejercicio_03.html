Contraste de Colores

1. Comprobación de Contraste

a. Texto color: #0000ff (Azul) y color: #5fff59 (Verde claro)

Estos colores no tienen suficiente contraste. Se recomienda usar un fondo de color claro como #ffffff (blanco) para el texto azul, o un fondo oscuro como #000000 (negro) para que ambos colores se distingan bien.

b. Texto color: #0000ff (Azul) y fondo: #ffffff (Blanco)

Este contraste es adecuado. El texto azul sobre fondo blanco es cómodo de leer.

c. Texto color: #1e05ff (Azul oscuro)

Se recomienda un fondo muy claro como #ffffff (blanco) o #f0f0f0 (gris muy claro) para asegurar suficiente contraste y buena legibilidad.

2. Test de color en la página web http://www.20minutos.es/

Se ha detectado que algunos textos en la página web tienen poco contraste. Por ejemplo, en algunas secciones los enlaces de color azul claro sobre fondo blanco no tienen suficiente contraste para personas con baja visión.

Es recomendable aumentar el contraste de los enlaces o cambiar el color a un azul más oscuro para mejorar la accesibilidad.
