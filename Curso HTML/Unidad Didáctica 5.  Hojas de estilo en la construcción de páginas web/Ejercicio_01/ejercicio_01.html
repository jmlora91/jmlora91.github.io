Subtítulo de la página

Un párrafo de texto como otro cualquiera.

a) Ubicar las reglas CSS de forma incrustada dentro del documento html.

        <head>
            <style>
                h1 {
                font-size: 50px;
                color: blue;
                font-family: Verdana;
                }
            
                h2 {
                font-size: 30px;
                color: red;
                font-family: Arial;
                }
            
                p {
                font-size: 20px;
                color: yellow;
                font-family: Arial;
                background: black;
                }
            
                section {
                margin-top: 30px;
                }
            
                pre {
                background: #f4f4f4;
                padding: 10px;
                overflow-x: auto;
                }
            
                body {
                font-family: sans-serif;
                margin: 20px;
                }
                </style>
            </head>

b) Crear un archivo CSS independiente y enlazarlo con el documento html.

        <head>
            <link rel="stylesheet" href="styles.css"> 
        </head>

c) ¿Cuáles son las ventajas de usar las hojas de estilo?

  • Separación entre contenido (HTML) y presentación (CSS).
  • Reutilización de estilos en múltiples páginas.
  • Fácil mantenimiento y edición del diseño.
  • Diseño coherente en todo el sitio web.
  • Mejora el rendimiento del sitio (caché de archivos CSS).

d) ¿Cuáles son las distintas formas en que se pueden utilizar las hojas de estilo?

  1. En línea: dentro de las etiquetas HTML con el atributo style.
  2. Incrustada: dentro de una etiqueta <style> en el <head>.
  3. Externa: en un archivo CSS separado enlazado con <link>.

e) Partes y sintaxis de una hoja de estilos

Una hoja de estilo tiene la siguiente estructura:

selector {
  propiedad: valor;
  propiedad2: valor2;
}

Ejemplo:

h1 {
  font-size: 50px;
  color: blue;
}
  • Selector: indica qué elementos HTML se van a afectar (ej. h1).
  • Propiedad: define el aspecto a modificar (ej. color).
  • Valor: es el valor que se le asigna a la propiedad (ej. blue).
