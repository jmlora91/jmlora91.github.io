d. Explicación de etiquetas y atributos necesarios para insertar imágenes

  • Etiqueta: <img>
  • Atributos necesarios:
    • src: Define la ruta de la imagen (puede ser relativa o absoluta).
    • alt: Proporciona un texto alternativo para accesibilidad y en caso de que la imagen no se cargue.
    • width: Establece el ancho de la imagen en píxeles.
    • height: Establece la altura de la imagen en píxeles.
  • Para convertir una imagen en enlace: Se utiliza la etiqueta <a> y dentro se coloca la etiqueta <img>.
