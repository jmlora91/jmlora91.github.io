Inserción de Imágenes y Elementos Multimedia en HTML

1. Imágenes

Para insertar imágenes en un sitio web se usa la etiqueta <img>. Los atributos principales que se utilizan son:

  • src: indica la ruta o URL de la imagen.
  • alt: texto alternativo que describe la imagen para accesibilidad.
  • width y height: establecen el tamaño de la imagen en píxeles (opcional).

2. Videos locales (archivos propios)

Para insertar videos propios almacenados localmente se usa la etiqueta <video> con estos atributos principales:

  • src: ruta o URL del archivo de video.
  • controls: muestra controles para reproducir, pausar, etc.
  • width y height: tamaño del reproductor de video.

3. Videos de YouTube (contenido externo)

Para insertar videos de YouTube o cualquier otro contenido multimedia externo se usa la etiqueta <iframe>, que permite embeber el video directamente en la página.

  • src: URL del video en modo "embed" de YouTube.
  • width y height: tamaño del reproductor.
  • frameborder="0": elimina el borde del iframe (opcional).
  • allowfullscreen: permite que el video se vea en pantalla completa.
